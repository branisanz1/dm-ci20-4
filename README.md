## Requirements

- [x] Php 7.1 or +
- [x] Make
- [x] Git
- [x] Docker
- [x] Docker Compose

## Installation

- Clone the repository
- Install dependencies: `make install`

## Usage

```
make start
```

## Launch tests

```
make test
```
